# Buuf Moon 1.0.0beta1

First beta release of Buuf Moon, a non-commercial Buuf-inspired complete theme for Pale Moon.

## Highlights

- Buuf-styled browser toolbar
- Buuf-styled Add-ons Manager, Preferences and Permissions icons
- unified khaki/green Buuf color palette
- custom tabs and scrollbars
- Linux and Windows visual fixes
