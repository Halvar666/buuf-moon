# Changelog

## 1.0.0beta1 - 2026-07-22

First public beta release candidate.

### Added

- Buuf Moon theme identity and Pale Moon complete-theme packaging.
- Buuf-styled main toolbar icons.
- Buuf-styled Add-ons Manager category icons.
- Buuf-styled Preferences panel icons.
- Buuf-styled Permissions page icons.
- Unified Buuf khaki/green color palette.
- Custom tab bar, active tab and inactive tab colors.
- Custom global scrollbar colors and narrowed scrollbar geometry.
- Global manager/dialog background styling.
- Linux and Windows-specific toolbar/scrollbar fixes.

### Changed

- Rebuilt the theme on a Camimoon-compatible Pale Moon structure.
- Replaced many inherited Camimoon/Mozilla icons with Buuf-derived artwork.
- Reworked hover and active states for toolbar icons.
- Recolored selected states, radio buttons, checked states and manager list selections toward the Buuf palette.

### Notes

This is a beta release. Further visual corrections may still be needed in secondary dialogs and rarely used UI paths.
